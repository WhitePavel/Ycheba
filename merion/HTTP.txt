HTTP - hypertext transfer protocol(протокол передачи гипертекста)
Протокол http передаёт гипертекст с гиперссылками , которые не просто ссылаются на что-то а представляют конкретный путь до другого файла в интернете.То есть его URL.
http - клиент-серверный протокол. Клиент,который хочет получить какие-то данные отправляет запрос серверу(http запрос) а, сервер выдаёт ответ(http ответ).
URL(Uniform Resource Locator) - унифицированный указатель ресурса

HTTP сообщение к состоит из 3-х частей
1. Стартовая строка
   (от пользователя)
   -URL
   -метод запроса
   (от сервера)
2. Заголовок
3. Тело сообщения
 
ответы от сервера
1хх - информирование о процессе передачи запроса
2хх -информирование об успешной передаче
3хх - информирование о перенаправлении
4хх - ошибки клиента - либо нет либо неправильно либо нельзя
5хх -ошибки сервера
