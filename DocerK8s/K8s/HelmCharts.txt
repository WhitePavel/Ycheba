Helm Charts - Helm - это менеджер пакетов для Kubernetes. Этот инструмент позволяет нам обернуть Kubernetes приложения в удобные пакеты, называемые чартами, которые можно легко развертывать, обновлять и управлять ими в любой момент времени

values.yaml - файл в котором мы обозначаем значения по умолчанию
Chart.yaml - технический файл в котором указывается инофрмация об этом чарте


helm list - показывает какие деплойменты через helm были сделаны
helm install *** ::: - поднимет helm
::: - путь
*** - название 

helm install *** ::: --set container.image=&&&&&& --set.replicaCount=3  - изменение контейнер имэджа
&&&& - имэдж контейнера
перезаписываем валуес
helm ugrade - обновляем уже бегущие helm с нужными параметрами
helm search hub apache
