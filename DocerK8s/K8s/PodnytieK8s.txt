нужно установить 3 приложения 
1.VirtualBox - нужен для создания виртуальноог сервера на вашем комьютере где будес создан K8s Cluster
2.Kubectl - для управления k8s Cluster'ом
3.Minicube  - для создания K8s Cluster Single-Node кластер из одной Node оптимизированный для тестов,рахработки и учёбы

minikube start - запуск minikube на локальном компе, при MACOS нам не нужно использовать гипервизор от VB так как оно будет привязано к докеру

Спасибо за урок )) На MacOS если уже стоит Docker виртулка не нужна - Кубернетис будет использовать Докер как виртуальную машину. При этом чтоб зайти как root достаточно docker exec -it minikube bash,   а если хотите подключение minikube ssh - запустит как docker пользователя))) Это важно тк бесплатного сервиса помоему уже нет для тренировокю

docker exec -it minikube bash
kubectl get componentstatuses - проверка роботоспособности
kubectl cluster-info - показывает ифно по кластеру
kubectl get nodes - показывает количество запущенных nodes
minikube stop - останавливает виртуальную машину, но сохраняет всё что мы запускали на ней
minikube delete - стирает всё в кластере

запуск с попределёнными параметрами ---------------

minikube start --cpus=4 --memory=8gb --disk-size=10gb
