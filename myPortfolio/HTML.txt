для создания файла мы создаём файл index.html 
внутри этого файла мы создаём теги, весть html создаётся из тегов
Например
<!DOCTYPE html> - указание 5 вверсии html
<html> - основной тег (открывающий)
    <head> - шапка названия(характеристики) странинцы,описание и т.д. всё это не видно для пользователя
      <meta charset="utf-8"> - характеристика поддержки кириллицы и латиницы  сам тег meta даёт ралзичные настройки для страницы	
      <title> Главная страница </title>
    </head>
    <body> - тег для пользователя
  привет
    </body>

</html> - закрывающий тег

основной образец


ОЧень удобно пользоваться просмотром кода в браузере гугл, оттуда можно брать стили для своей работы
