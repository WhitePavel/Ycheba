title="*****" - атрибут позволяющий выводить подсказку для того или иного текста
Применяется ко всем тэгам
-----атрибуты для <ol>-----------
<ol start="*"> - указывает с какого элемента будет начинаться список -только для списков 
<ol type="a"> - тип по алфавиту
<ol type="i"> - тип по римским цифрам
-------атрибуты для ul-------
<ul type="circle">
<ul type="square">

у одного тэга может быть неограниченное кол-во атрибутов

------------СОЗДАНИЕ ССЫЛОК--------

<a></a> - Тег для созлания ссылки в языке HTML
для данного тега обязательно нужно добовлять атрибут href
<a href="https://google.com">Google</a>
