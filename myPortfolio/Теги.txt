<!DOCTYPE html> - тег указания последней версии html
<html> </html> - тег для создания html файла
<head> </head> - тег указывающий технические характеристики страницы

<meta ***> - тег указания настроек страницы (пишется в HEAD) - описывают только конкретную страницу
<meta name="author" content="Павел Просветов" нужны для поисковых роботов
<meta name="description" content="Тестовая">
<meta name="keywords" content="тест,проверка,html"

<title> *** </title> - тег названия страницы в браузере (пишется в HEAD)

<body> *** </body> - тег для информции выводимой пользователю

------------работа с текстом---------
<h1>******</h1> - заголовок для сайта от большего к меньшему
<h2>******</h2>
<h3>******</h3>
<h4>******</h4>
<h5>******</h5>
<h6>******</h6>

<p>Просто текст</p> - просто текстовые надписи, каждая с новой строки
<p>Просто текст</p>
просто текст - данные текстовые заголовки будут выведены в одну строку
просто текст - т.к. для HTML нет переноса строк в данном примере

<em>текст курсивом</em> - здесть текст выделяется курсивом
<em></em> == <i></i> - тот же курсив

<b>текст тут выделяется жирным </b> - текст выделяется жирным
<strong></strong> == <b></b - тот же жирный шрифт

<strike></strike> - тег,делающий текст перечёркнутым
<sub></sub> - тег,смещающий текст ниже основного ряда
<sup></sup> - тег смещающий выше основного ряда
<br> - тег переноса строчки на новую линию
<hr> - тег создающий горизонтальную линию


----------обозначающие теги------
<abbr title="HTML">HTML<abbr> - тег для обозначения аббривиатур
атрибут title позволяет указать тот текст,который будет появляться при наведении на объект

<address></address> - тег,добовляющий стиль курсивом для адреса откуда

<big></big> - тег увеличивающий символы
<small></small> - тег уменьшающий символы

<pre> тег для сохранениея всех абзацов и пробелов </pre>
<code> </code> - указывается монаширный шрифт
