----------------------------------------Создание репозитория-----------------------------------------------------
1.git init -инициализация репозитория
2.git add --all - добавление всех файлов репозитория в staged(подготовленный)
3.git remote add *** $$$$ - связывание репозитория с удалённым на гитхабе
**** - имя
$$$ - url адресс репозитория
4.git commit -m "first commit" - коммит файлов
5.git push -u **** main - запушивание файлов в удалённый репозиторий
**** - имя репозитория
main - корневая папка
git remote -v -проверка связи между гитом и гитхабом
