Если создал какой файл и уже добавил его в список на изменение с помощью
git add но потом передумал убрать его поможет команда 

git restore --staged *****
**** - файл который хочешь убрать
restore(англ.Восстановить)
что бы убрать всю папку из состояния staged можно сделать
git restore --staged . (она сбросит текущую папку)

можно откатить коммит на более раннюю версию файла с помощью команды
git reset --hard ****
**** - хэш коммита длинный или укороченный через git log --onelineø

Если файл был случайно изменён и нужно вернуть его в исходное состояние то 
тут поможет команда 
git restore ****
**** - файл который нужно откатить до изменения
