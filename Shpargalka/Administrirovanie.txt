Для локального репозиторя
--------------------------
git status - проверка текузего статуса в папке гит

git add --all - добовление всех файлов в подготовительное состояние к коммиту
add - добваить
all - всё

git diff -команда позволяет посмотреть изменения последнего закомиченного файла
с текущей версией файла.

git diff --staged - что бы посмотреть файлы добваленные через add

git commit -m "*****"
**** - коментарий к коммиту
-m(флаг)(messege) - указывание сообщения коммита
--------------------------
Получение инофрмации от предыдущих коммитов
git log - посмотреть историю коммитов

git pull - вытянуть с удаённого репозитория всё в локальный.
